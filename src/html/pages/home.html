Партнерская программа для разработчиков веб-приложений

Что такое Дневник.ру:

  • Cоциальная сеть для школ

    Персональные профили, группы, события, учительская/родительская, игры и приложения, лента новостей, образовательный и пользовательский контент и многое другое.

  • Электронная школа и гос. услуги

    Веб-сервис для школ и автоматизированные информационные системы для органов управления образованием.

  • Дистанционное обучение

    Платформа для видеоконференцсвязи и совместной работы, электронные образовательные ресурсы и учебники, тесты/опросы и пр.

Центр приложений

В Дневник.ру реализован Центр Приложений,
куда интегрированы как образовательные,
так и развлекательные веб-приложения.

Посмотреть подробную презентацию

Размещение в нашем Центре Приложений позволит вам:

  • Работать с аудиторией
    5+ млн. пользователей
  • Использовать API для получения информации о пользователях и персонализации предложения
  • Бесплатно воспользваться маркетинговыми
    возможностями Дневника

Основные технические требования

  • соответствие возрастному цензу 6+
  • интеграция через iframe
  • работа только по протоколу https
  • максимальный размер фрейма 940*1200 px, рекомендуемый - 940*600 px
  • API для разработчиков

Документы

  • Описание для партнеров полные ТТ, рекомендации по разработке,

  • Образец договора
